Rigåvur
Įe övdalsk sågå
Fuost kapittleð
Iessn launggu war eð twär kullur, so ietteð Ieðwig og Elį. Dier war ått år og skaulkamrater. Wenn dag läkt dier min weroðrum snjäst skauln add slutað. Mjäst åv war dier aut og läkte.
Um wittrą rennd kullur ą̊ skaiðum eld fuor ą̊ spartj i lag min krippum i bymm. Dier brukeð sai åv fer mun og faðer um ukum sturum skaiðbokkum dier dugd fårå ni’tter autą̊ tä foll umringg, eld ur launggt dier add ferið ą̊ spartje. Milumað sett krippär iuop sparką og uok slais og ien tu ni’tter laungbokkum frą̊ bymm. Eð flog tįstrur frą̊ sparktuę, dar dier add faið upp fartn. Krippär kuleð so eð lämäð launggt ini raiseð. Ą̊ so såt framą̊ endam styrd min ien spark fråmånað sig.
Fuotjeð festeð i lampur autą̊ bru’nn snjäst eð byrd ą̊ werd mörkt. Eð lyst liteð ą̊ trampur og ą̊ sniųo’nn niðånað. Eð lupteð rätjem, fer oller i bymm add wiðåspisn i tjyötję.
Ienn witterkweld framter að juolum bai’dd fuoreldrär Ieðwig og Elin umm kullur uld kum iem. Dier bai’dd fą̊ är krippą autą̊ bru’nn og ur dier uld kum inn i tjyötjeð roðer um ogų og sai »Gusin!« og setts sig upą̊ ien stuol og knåyt upp sny̨ ̨örę ą̊ pjäksum og spienn åv sig diem.
»Friuos du um fuotą?« brukeð munur ðieras spyr etter. »Fåmm kenn ą̊! Uą ur kold du ir! Set ą̊ ðig luddur!«
Framą̊ ettermidaem add Ieðwig og Elį ferið ą̊ spartj dai’tter ienum we so bar åv ini raiseð. Kullur glemäð og skwäpäð, og að slutę wart dier i wari dier wart wend att og far iemat. Men eð war it wegen iemat. Dier kenndes it wið sig!
»Wert far isin wegen? Wendum att og farum að oðer oldę«, sagd dier wið weroðrer.
Eð fuor blås og wart aurweðreð. Blästn tuog i mįer og mįer. Sniųo’nn danseð ringgum kullur. Trainę rukkeð og kryökt sig. Blästn kuleð, sniųo’nn lagd sig i stur drivur autą̊ weem. Kullur war tungner yöp að weroðrum, ellest edd dier įt að ärt noð wenn dier sagde. Liuotweðreð kam uvendes strai’tt og ųodält.
»Wert ulum wįr fårå?« spuord Elį etter. »Kumum older iem noð mįer!«
Kullur fingg painas og klå’ss min rigåvum, mes dier boð til tågå sig framm gainum sniųo’nn. Dier druog nið ärlappą ą̊ ettum og kneppt iuop bandę under akun og druog upp dragtjaiur ǫ jäkkum so launggt dier dugd upi åsn. Dier fros uvliuo’tt um nevą̊.
»Nų lärum wįr fel friuos ijel«, lit Ieðwig.
»Settsum uos under ien grån«, sagd Elį. »Bellum troðå til iet uol i sniųo’mm, dą̊ werd eð slais og weggär ringgum uos.«
»Näi, so beller an it fårå«, miend Ieðwig. »Kumum tä såmmen og werd attrumsni’dder, wet ig. Indjin kumb tä finn att uos. Eð kumb grą’ll tä laiken sniųodrivum dar sittsum.«
Elį sett sig upą̊ spartjin og lit ą̊ fikk luv wail sig ien stjyöru. Blästn kuleð og sniųo’nn fuor ringgum diem. Elį såmneð upą̊ spartjem, nikkeð til og wakkneð att, men ą̊ såt kwer dar ą̊ såt.
»Ärer du?« twerspuord Ieðwig. »Eð låt sos įe stjälla.« Elį ärd ą̊ og. »Ir eð frą̊ ienum eststjuo’ss? Ukin ir auti raisę min ien est i issu weðrę?« Elį rai’tt upp sig. Kullur wend sig mųot blästem og yöpt mjäst dier dugde: »Ööy! Kåm jųot og jåp uos! Oldum ą̊ työlå!« Elį sett sig upą̊ spartjin atten. »Du fą̊r it sitts kwer upą̊ spartjem«, sagd Ieðwig. »Snjäst an friuos werd an brågå sig, lär du
fel witå!« »Ja, naug wet ig fel eð«, suoräð Elį og stelld sig brieðwið spartjin. »Men fari dą̊!« sagd Elį og lit fren.
1
»Ig will it.« »Du fą̊r luv!« Läteð so ärdes ini raisę kam nemmer. Kullur wend sig mųot liuoðę.
Oðer kapittleð
Fråmånað diem stuoð ien stur est og ien sliði min ien gamtkall i. Kalln add ien sturkasungg og ien stjinnett so ną̊’dd niðą̊ enneð. Yvyr kni’mm add an ien stjinnfeld.
»Uą, ukk krippwasker!« lit an mes an fikk sją̊ diem. »Nų irå rigåvur aut og fårå liuo’tt min ið, kanenda. Dar dier irå riktut ą̊ taę og slait i oll so ir ollsta’ss, al an dą̊ it war aute. Og snjäst dier drågå minn sig sniųo’nn og koldan, werd eð fa’llit. Settsið ið niði sliðån og fyið mig iem að mųor og wermið ið i stugun! Byddjum ini raisę, grą’ll jųotånað jär.«
Kullur stuoð kwer upą̊ weem. Dier fros og war grą’ll stierder. Kalln kliev autyr sliðåm og lypteð fuost upp Elin og sę Ieðwig og sett diem attåni upą̊ ien feld og brie’dd ą̊ diem ien eller feld. Sparką tuog an ą̊ släp.
An smakkeð að estem og wend att að dyö oldę weðą̊ an add kumið. Estn djikk lyngnt og ruoli mųot rigåvum, og stjällą lit. Sniųo’nn danseð ringgum sliðån sos nog wait slöyer.
»Farir ieðą̊, liuotans rigåver!« yöpt kalln. »Farir iemat daiti Rigåvbjärr, dar kumir frą̊!«
Kullur såmneð niði sliðåm. Mes kalln, so ietteð Walle, wild glåmå min diem, suoräð dier it noð. »Eð war nųfel tið på finn att diem«, ugst an innum sig.
Trið kapittleð
Kelindję Walla, Tuova, djikk att og framm iemi tjyötję og bai’dd etter gubben uld kumå min kullur. Tuov add ien kristallkaul. I enner dugd ą̊ sją̊ ollt so fuor framm auti raisę. Ą̊ add fel uorteð i wari Ieðwig og Elin i kristallkaulun og si’tt ur dier paindes min rigåvum. Ą̊ såg sniųodrivur autą̊ weem, ą̊ såg kullur wart trä’tter og trä’tter, og ą̊ såg Ielį sett sig upą̊ spartjin og såmneð.
»Kringg dig auti raiseð og jåp krippum«, add Tuov sagt að kallem sainum Walla. »Kringg dig föreld rigåvur djärå kullur aiskolder og finggrę og töner werd að aisbitum. Snąrt uork dier it old ą̊ painas laingger i iss aurweðrę!«
Dar Wall add sett ą̊ sig warrmą̊lsbruok, kasungg og stjinett og ullsukkur niði stjinnstyvlum og sę digerulluottą, fuor an daiti stolleð og sagd að estem:
»Nų amm wįð brå’tt umm, um ulum winn finn att diem auti raisę föreld rigåvur åvå gart aisn åv diem. Du kann fel we’n, fast an ir attrumsni’nn. Nų laikter ig ą̊ dig so kumum framm að kullum!«
»Welest estn fann att kullur!« ugst Wall mes dier add kumið framm að stugun ini raisę. An lypteð upp kullum yr sliðåm og bar inn diem ini tjyötjeð.
Tuov tuog mųot åm og lit an uld legg kullur i iendier saindję ðar.
Fiuord kapittleð
»War irum wįr?« lit Ieðwig mes ą̊ wakkneð. »Irum i ien stugu ini raisę«, lit Tuova. »Ig såg i kristallkaulun ur paindesið min rigåvum. Men
nų irið friðåðer nest uos. Eð ir fel wildest stą’n jär tast liuotweðreð ar ferið. Ig sir i kristallkaulun mun og faðer irå brygder war irið. Ulum stjikk buoð min kuorpin menn, Ugå, daitað diem.«
2
Tuov åp kullum tag åv sig jäkkur, skaiðbryökär, ettur og uottą og latt diem fą̊ lån ulltråyur Walla. Ą̊ stelld tųo stuol fråmånað yppenspisem, dar eð war bråsą̊, og sagd að krippum setts sig dar og båså sig. Dier tuog åv sig pjäksur og fingg lån åvsukkur og luddur set ą̊ sig.
»Ig al sją̊ etter um ig ar tau papir og twär blyaspennur, so bellið skriev elsninggär að fuoreldrum iðrum«, sagd Tuova.
Ą̊ sett sig för sig siuov og glemäð tyst wið kristallkaulų, mes ą̊ snurreð ån runt, runt i nevum. Ą̊ lit inn i kaulų Ieðwig og Ielį mą̊’dd wel og war nest frekåfuotję.
Kullur og Tuov åpes að skriev ą̊ papirum: »Gusin, mun og faðer! Mą̊um wel nest Tuov og Wall i stugun auti raisę. Wall al stjuo’ss iem uos min est og sliðå snjäst liuotweðreð ir yvyr.« Lainggst nið ą̊ papirum skrievd dier namnę sainų. Tuov wiek iuop papirę og stuppeð nið diem i smą̊kuppą ą̊ add gart åv takkstjinnę.
Ą̊ knåy’tt ien tunn stjinnriem ringgum kuppą, lindeð ringgum sig ien sturan sjal og fuor autą̊ brunę min stjinnkuppą i nevum og yöpte:
»Ugi, Ugi, Ugii!«
Dar li’tt add werið kam ien stur ien kuorp og stelld sig upą̊ iendier erdę Tuovu. Ą̊ sagd að kuorpem an uld daiti bynn og slepp nið stjinnkuppum autą̊ dörär dar kullur byggde. Ugi tuog stjinnkuppą i kruksn og flog umstað.
Femmt kapittleð
Iemnest Ieðwig sleppt kuorpen nið kuppam min lappin að mun og faðer og krakseð flier gaungg. Sę flog an iemað Elin og djärd laikt. Föreld an fuor atter daiti raiseð flog an daitum auseð iessn og krakseð.
Fuoreldrär kullum undres ą̊ wenn eð war so lit so og fuor aut. Dier fann att stjinnkuppą og fuor inn min diem. Tuov såg fuoreldrą ini kristallkaulun og lit tyst: »Ev upp stjinnkuppan og sją̊ etter wenn so ir inni!«
Ą̊ såg fuoreldrär syöks däler mes dier add lesið ą̊ papirę under lampun ini tjyötję, og ą̊ sagd åv dyö fer kullum. Tuov, Wall og kullur sett sig nest buordę og åt suoðwattneð og skarpbroðsgą̊sär min smyöreð og smą̊uostn so Tuov add gart. Dier glemäð um sniųo’nn og rigåvur dar aute.
Mes Ieðwig sagd åv Elį add wilað far inn under ien grån og setts sig, lit Tuov rigåvur add twisåð að ener ą̊ uld fårå so. »Dier irå so illrokker og itt ą̊ so mitjið dar fuotjeð ir aute. Dier fårå ringgum uos sos dier edd að sagt: »Ulum wįr läka?« Men dier åvå įt lag ą̊ eð werd uvkolt að ien menistj.
Um kweldn, mes dier add gaið og lagt sig i sainggär i sturtjyötję, ärd dier ur rigåvur lit dar aute. Ieðwig og Elį låg brieðwið werr oðrer i ien saingg i iendier rå’nn og kuogäð ą̊ brasų daiti yppenspisem feld dier såmneð.
Sjäkst kapittleð
Mes Ieðwig og Elį wakkneð um morgun oðer da’n war ðier įesumer ini tjyötję. Ien mass låg og sov i fuotendam ą̊ saindjin.
Frą̊ lisslkuvåm brieðwið ärdes noð liuoð åv noger. Eð war noð so knäpäð lit fesiktut. Elį kliev yr saindjin og fuor autą̊ iet liuoð daitað dörum og bögles inn i kuvån. Wall såt nest ien buord so war fullt i gräjum og jält ą̊ lågå ien pjäksu. Eð war pjäksą Elin. An sluo i pliggą og festeð suolų ą̊ ienduer saiðun. Elį war upi ferunder yvyr ollt ą̊ såg upą̊ buordę – eð war boð ambrär, sylär, ną̊ lär, biktrå’n, ien litn plåtask min noð åv noger niði, stur leðerbiter og įe stur og įe litę wiðålest. Guoveð war kolt. Elį djikk atter daitað Ieðwig i saindjin.
3
»Wall lägär pjäksun mener so war sund«, twisäð ą̊ og lagd sig brieðwið Ieðwig.
»Ig undres ą̊ war Tuov ar kristallkaulų noger«, sagd Elį. »Ig edd wilað witå um ig dug sją̊ noð inni.« Kullur kuogäð ringgum sig i tjyötję, men dier såg it ingg kristallkaul.
»Ą̊ ir naug ini skåpę dait dan«, sagd Ieðwig og piekeð. Men dier wild it umstað sją̊ etter, autą̊ tä glåmå min Tuov fuost.
Dier ärd Tuov autą̊ bru’nn. Ą̊ prikäð åv sig sniųo’nn. Mes ą̊ tept upp dörum inað fösstövį ärd kullur ur eð liuotblis dar aute. Rigåvur tuog tag i dörär, og sniųo’nn blis inn.
»Jär kumir it inn noð, rigåvur!« yöpt Tuov mes ą̊ tept att åy’tterdörum og rak att lą̊stie’nn i kabblą̊sę. Ien stur iemsmai’dd nytjyl fikk war kwer i lą̊sę ą̊ autsaiðun.
»Lęs upp dugir dą̊ įt«, iembleð ą̊ fer sig siuov.
Tuov kam inn i tjyötjeð min ien mjokbytt. Ą̊ add werið auti fją̊sę og motjeð tjynär og ųogås takkum. Ieðwig og Elį stieg yr saindjin og sett ą̊ sig. Tuov elld lit åv nysilaðmjok daitą̊ iet liteð fat að kattem, föreld ą̊ byrd ą̊ duon min braddun.
Wall kam frą̊ lisslkuvåm og sett sig nest buordę. Kullur åp Tuov duon ą̊. Dier åt welindjin og blotbroðsgą̊sär min iemtjinnaðsmyöreð, gambeluostn og myssmyöreð attrað.
»Welest amm tjynär so amm noð jätå«, lit Tuova. »Ukað åvå dier tjynär i ollum gardum iemi bymm?« spuord ą̊ etter.
»Ja, oller autą̊ lärinną i smą̊skaulam«, suoräð Ieðwig.
Kullur sagd að Tuov dier edd uvendes djienn wilað fą̊ old i kristallkaulun. Tuov rai’tt sig og fuor daitað skåpę og tuog framm åna. Kullur jält i kaulun wer sain stjyöru. Dier rwieð en runt, runt, men dier dugd it sją̊ noð inni.
»Indjin otą̊ ig dug sją̊ noð ini kaulun«, suoräð Tuova. »Settsið ið attrað mig, so will ig jåp ið sją̊ wenn munur iðrer djärå fer noð.«
Sjuund kapittleð
Tuov tuog kristallkaulų min båðum nevum, wend og rwieð en runt, og min dyö sumu wart eð liuost ini kaulun. Kullur såg munur dieras jält ą̊ stell til juolum. Mun Ieðwig bäkäð bullą og mun Elin djärd pipärkakur i lag min Wiuolu, stursyster Elin. Dier såg stjinnkuppą so Ugi add flueð dait min og.
»Nų lainggtis fel iem, mes sją̊ ið ur dier old ą̊ old ą̊ stell til juolum iem iðer«, lit Tuova. »Ulum fel upås rigåvur fårå ieðą̊ nossn, so tagið ið iemat! Welest luvdaär åvå byrdas nų mes juolär irå.«
Kullur lit dier tykkt underlit dier add įt older ärt tålås um Wall og Tuov og stugų auti raisę. Indjin add it glåmåð noð um diem iemi bymm.
»Eð ir fel it eð underlit noð«, lit Tuova. »Wilum elst įt råk noð sturt fuok. Men krippą wilum wįð råka. Um sturfuotjeð kumå uvnęr uoss eld stugun, djär ig uos ųosynligų. Um dier kumå uvnęr tjy’mm og takkum ųorum dar dier go i raisę og jätå um såmårn, djär ig krytyrę ųosynligų. Ig sir i kristallkaulun snjäst menistjur kumå uvnęr uos. Indjin otą̊ skuogwakterfamilln so byddjer laingger ini raisę niðåni Rigåvbjärr kumb inn jär. Dier irå iend grannär amme. Jåpum weroðrum min oll so ir. Dier sai older åv för nogum um uoss, og ig jåp åm dar familjem min tä djärå an ųosynligan snjäst dier wil so. Um kuorpen Ugi kumb og krakser triųo gaungg autą̊ bru’nn, wet ig wenn eð bityðer: Eð ir menistjur i raisę og grannär wil werd ųosynliger.«
»Rigåvur lärir fel war redder, boð ir og grannär, ę?« lit Ieðwig.
»Ja, um dier yvyrrask uos og kumå stra’itt, kringgum wįr uos iemat. Wilum int noð djienn wårå auti raisę dar dier irå i fartem. Men um sumbrą irå dier wið fri’n mjäst åv. Dą̊ bellum wįr wårå auti raisę og finn berę autą̊ rigåvur rust min uos, sagd Tuova. Dar eð raingner kunn rigåvur old sig framm og läk min muolnum og rekå iuop diem og djärå skaurmannsmuolnę åv diem. Sę diel dier muolnę atten og flytt diem og djärå åskmuolnę ą̊ flierum stellum. Men rett sos eð ir

luss dier lie’ssn ą̊ tä djärå digermuolnę og blås i tau diem og slepp framm suoln. Sę fårå dier atter upi Rigåvbjärr og wail sig.«
»Snjäst eð werd ostn werd rigåvur braguger og brygder«, sagd Walle. »Dą̊ fą̊ dier luv fårå nån weg. Dier fårå frą̊ Rigåvbjärr. Dier rust min lovę og dans umringg min dyö, lypt upp dyö og slepp nið dyö, lypt upp dyö og slepp nið dyö. Rigåvur jag loveð og flytt dyö launggt brott og dans min dyö atten. Dar mikkel rigåvur irå ą̊ sumu stell kunn dier wårå so starker so dier briuot åv trainę. Eð ir eð windfellę järi raisę, sturgrånär og sturtollär so åvå uolteð og ligg min ruotur upi weðrę. Dar dier åvå stendeð ir eð įt bar įe stur įe grav. Men oll wesst min rigåvum ir eð dar dier fårå runt, runt og flytt sniuo’mm um wittrą og stell til so įt an beller far aut, fer įt eð gor sją̊ noð fråmånað sig. Dą̊ råð dier, då irå dier starkestär. Dar blästn kuler ringgum stugų gleð rigåvur upp sig.«
Wall rai’tt upp sig og djikk daitað yppenspisem og add ą̊ mįer åv wiði. Iedwig twisäð að Elin: »Ar du tokkað Wall fer an lägäð pjäksun dener?« »Näi, dyö ar ig grą’ll glemt åv!« »Gok daitað åm nų!«
»Tjär tokk fer ðu wild djär að pjäksun mener«, sagd Elį að Walla.
»Eð djärd ig fel djienna. Ig djär skuoną og, boð að uoss og að grannum ųorum daiti raisę, og ig lägär skuo’mm að uos ollumiuop. Grannär åvå sjäks krippa, so eð rekk til lågå olltiett.«
Åttund kapittleð
Fråmånað iendier glasį ini tjyötję stuoð ien spinnrukk min ien stuol brieðwið. Ieðwig piekeð ą̊ spinnruttjin og sagd að Tuovu: »Wais uos ur an far dar an spið spunån!«
Tuov tuog upp ka’llaðullę frą̊ ienum kordj nest weddjin. Ą̊ sett fast ullę upą̊ ien stelningg upą̊ spinnruttjem, sett sig upą̊ stuoln, druog i gaungg sturjuoleð min ienumdier nevåm, trampeð ą̊ ien tramp og byrd ą̊ spinn spunån. Ą̊ tuog ka’llaðullę min finggrum og latt ullę spinnas að ienum tråð so ty’lld upą̊ ien eller stelningg.
»Föreld an ar kumið so launggt, fą̊r an luv ka’ll ullę, so ą̊ werd rien frą̊ tuorpę og werd grann og blot sos ullę niði kordjem. Eð ir sakt mitjið so al djärås feld an ar faið spunån so an beller djärå gerdę min. Kunnið djärå gerdę?«
»Ig ar fel buoðið til«, sagd Elį, »men eð ir wenest knevlut.«
»Um bar an yöver sig, gor eð fel wildera og wildera. Bellið kum jųot att að wårn og sją̊ dar klippum takkur. Wall bruker old i diem, og ig klipper fel diem min ien stur soðsaks.«
Eð ärdes noð liuoð autą̊ bru’nn. Noger rwieð runt nykklem i lą̊sę og fuor inn i fösstöveð. »Nų kumb Mas«, lit Wall mes eð knakkeð ą̊ dörum inað tjyötję. »Jaa. Kåm inn«, sagd Tuov og Wall min dyö sumu.
Niund kapittleð
Inn kam ien litn gubb i ienum sniųougum kasundj so ną̊’dd niðað fuotum. An tuog åv sig ien stjinnett full i sniųo og lit strai’tt: »Nų fą̊ið luv jåp til! Lisslkullą, Tultą, ar fesuonneð. Ą̊ war aut og läkt mes rigåvur kam og fuor framm og remd og skremdes. Ą̊ kåy’tt naug og uld kum undą̊ og låg niðer. Ukin wet war ą̊ ir noger? Amm gaið i raisę og lie’tt og yöpt ›Tultą!‹ men amm įt dugåð finn att åna.«
Ieðwig og Elį war grą’ll tyster og stuoð og kuogäð ą̊ Mas. Lisslgubbin min launggkasundjin add wai’tt år so ną̊’dd niðą̊ erdär, launggt wai’tt skegg og roðų andlit. An lit: »Wilið fy mig upi Rigåvbjärr og sją̊ um rigåvur åvå Tultų iem sain? Bar kumum inn dar! Ig truor eð ir it tä went sig dier tep upp dörum fer fuotję.«
5
»Fyum dig dait!« lit Wall og Tuov min dyö sumu. »Welest Ieðwig og Elį irå jär. Ig wet rigåvur ev upp dörär i Rigåvbjärr bar að krippum«, sagd Tuova.
Mas fuor daitað kullum og spuord etter um dier wild fy daiti Rigåvbjärr og biuoð til fą̊ aut Tultų. Ieðwig og Elį lit dier wild sakt fya, og dier byrd ą̊ set ą̊ sig åy’tterklautą rað weg.
»Dier ðar klautär rekk dą̊ įt til noð i iss weðrę«, lit Mas. »Bellið fy mig iemat og lån klautą min krippum mainum – kasunggą og warrmą̊lsbryökär og stjinnettur og ulluottą. Dą̊ klarið ið niðunder feldem i sliðåm.«
Wall og Tuov add lag ą̊ wenn dier uld åvå fer klauta.
I twemm sliðum bar eð åv iemað Masa. Krippär os latt lån aut klautą saina. Kelindję Masa, Nimåna, add klautą fråmåni Ieðwig og Elin, kneppt att knįepur og stuppeð niði sjaln innåni kasundjin og jält upp uottą fråmånað diem, so eð war bar rek niði nevą̊. Nimån gav Mas klautą an uld set ą̊ Tultų. Syskunę war tyster og stuoð og såg ą̊ mes dier stelld til sig.
I sliðåm Mas fygd sturbruoðer Tultun, Akke. Ann war fiuortą̊ år. I oðer sliðåm såt Wall, Tuov, Ieðwig og Elį. Tuov add kristallkaulų minn sig i ienum lisslum kordje.
»Fą̊ið luv winn iem föreld eð werd mörkt«, sagd Nimån mes dier druog til. Ą̊ kolld ą̊ Ugå, og bai’dd an uld kumå. Ann kam og sett sig upą̊ iendier erdę Nimånu, og ą̊ sagd að åm:
»Fy ą̊ sliðum gainum raiseð og ųogäs fuotję. Um noger sliði far sund eld noger djär ill sig, fą̊r du luv kum jųot og kraksa. Dą̊ far ig etter min launggsliðåm og oðer estem amme og jåp dyöm.«
Estär add it stjällur ą̊ sig, so rigåvur uld it är sliðir kam upter Rigåvbjärr. Liuotweðreð indreð it estum tä tågå sig framm auti raisę. Ruoli og styö’tt djingg dier gainum sniųo’nn. Kuorpen fygd ą̊ ien bit attånað.
»Nų fą̊ dą̊ rigåvur djävå sig«, lit Walle. An wend ą̊ skollan og yöpt að kullum so war attåni: »Friuosið?«
»Näi!«, yöpt dier.
Tuov lypteð upp kristallkaulun upyr kordjem. Ą̊ jält i en min uottą ą̊ sig, men eð djikk it noð wel wend runt ån min uottum so war sniųouger, og ą̊ tuog åv sig dyöm atten. Feld ą̊ wart uvstierd um finggrę wann ą̊ sją̊ ien lissl kull ini kaulun. I ienum sturum sal ini bjärrę danseð rigåvur ringgum kullų. Dier war dier laungger og mjåer og swipäð min noger åv noger og wild kullą uld dans min diem. Ą̊ wild it ą̊.
»Dier sją̊ aut gainumstjinliger«, sagd Tuov wellt fer sig siuov. »At įt dier bigrip ien litn kripp int traivs i kuldam inn nest dyöm i ienum sturum stįesal ini bjärrę!«
»Glemä įt fer dig siuov. Seg åv wenn du sir!«, lit Walle. Tuov sagd åv mes ą̊ sett sig upą̊ uottą.
Wall lit »ptroo« að estem, og an stą’neð fel. Mes sliðin Mas add kumið dait og stą’nað, sagd Wall åv wenn Tuov add si’tt ini kristallkaulun.
»Au au au«, lit Mas. »Werdum kringg uos! Tultą ir nest rigåvum ini bjärrę!« Sę yöpt an að rigåvum so war auti raisę: »Ur avir ferið? Bigripir it ien litn kripp beller it byddj i kuldam iemnest ið!«
Wall og Mas smakkeð að estum. Dier byrd ą̊ go gainum sniųo’nn atten. Sturgrånär rukkeð att og framm i aurweðrę. Sniųo’nn räsäð frą̊ kwistum. Sliðin Mas wart full i sniųo. Akk yöpt daitað sliðåm so war fråmånað, men dier ärd it noð og fuor framter.
Įe sturgrån fjäll grą’ll fråmånað sliðåm Wall og lagd sig i we’n. Estn twerstą’neð. Wall kliev autyr sliðåm. Mas og Akk syntes it til. Wall wende, og mes an add ferið ien bit, fikk an sją̊ Akk mukäð undą̊ sniųo’nn yr sliðåm.
»Rigåvur biuoð til inder uos«, sagd Tuova. »Itjä dugå dier dą̊ slaikt«, lit Walle. Ugi såt upi ienum tolltull og såg Wall og Akk mukäð sniųo’nn. Kuorpen fygd ą̊ fuotję sos an
add uort tilsagd. Sliðir fuor framter mųot Rigåvbjärr. 6
Dier fuor ringgum grånę so add lagt sig i we’n. Tuov tuog åv sig uottą og add upp kristallkaulų upyr kordjem att. Ą̊ såg i kaulun sturdörär i Rigåvbjärr fuor upp og nog rigåvur wart innsleppter. Dier so war inn dar tuog mųot diem. Tuov såg dier stuoð nęr weroðrum min skollą ti’tt iuop. Ollt ollt iett brägäð dier sig runt og runt.
»Dier irå brygder yvyr noð åv noger«, ugst Tuova. »Rigåvur so åvå weð aut sai åv noger. Dier lär fel glåmå um uoss og sai åv irum ą̊ weg gainum raiseð. Tultą kuogär ą̊ rigåvur min sturum ogum. Ą̊ kąstji bigrip lietum ån.« Sę sagd ą̊ för sig siuov: »Ig al ugs umm ån liteð og sai inn i kristallkaulų irum ą̊ weg eter ener, og irum upi Rigåvbjärr um ien stjyöru.«
Ą̊ add djärå naug glåmåð rieð föreld sliðin twerslaungg til og war rieð welta! Tuov, so war uottlos og fros um nevą̊, tappeð nið kristallkaulun i diuopa sniųo’nn. Ą̊ yöpt til: »Kristallkaulą!« Ą̊ kuogäð niði sniųo’nn og lit: »Ig tappeð kristallkaulun! Ą̊ ligg dan niði.«
Akk uppeð yr sainum sliðå föreld Mas add faið til an stą’n. Akk uoðäð gainum sniųo’nn og sagd að Tuovu: »Ig al sakt finn att kristallkaulų!«
An stelld sig ą̊ kninę og kennd etter min ienumdier armem i sniųo’mm so war ä upað erdum. Að slutę yöpt an: »Ig kenner kaulų! Jär ir ą̊!«
An wart tag i min båðum nevum för tä fą̊ upp kaulų. An gav Tuov åna. Ą̊ kringgeð sig legg nið ån i kordjin og lit: »Welest du fikk i kaulų! Tjär tokk fer!«
»Rigåvur biuoð til inder uos liuotest dier dugå«, sagd Walle. »Ig werd djärå uos ųosynligų um ulum kum framm!« lit Tuova.
Dier stą’neð, og Tuov yöpt að diem so war attåni: »Nų djär ig uos ųosynligų fer rigåvum!« »Welest!« yöpt Akk liuotest an dugde.
Tiund kapittleð
Mes dier add stą’nað fråmånað sturdörum i Rigåvbjärr, stieg Ieðwig og Elį yr sliðåm og fuor daitað dörum og dunäð ą̊ dyöm liuotest dier dugde. Tuov add it gart diemm ųosynliger fer rigåvum. Ą̊ wenteð dier kam tä tepp upp snjäst dier såg twer kripper dunäð ą̊ dörär dieras.
Ieðwig og Elį sluo min knåytnevum ollt dier dugde, men intnoð ende. »Itjä wil dier tep upp noð«, sagd Akke.
»Fåmm luv baiða«, lit Tuova. Ą̊ såg i kristallkaulun rigåvur wart ųoruoliger. Older i wärdn add nogų menistj sagt að dyöm tep upp.
Ieðwig og Elį fuortsett dunå ą̊ sturdörär, og að slutę remd dier end autyr: »Tultą! War it rä’n, Tultą! Kumum etter dig nų. Du al fą̊ kum iem!«
»Eð ir krippär aut dan«, sagd Tultą wið rigåvur. »Teppir upp! Dier wil kum inn.«
»Ir eð krippär aut jär?« spuord rigåvur etter. »Dier fą̊ fel djienn dier kum inn að uoss. Tyttjum fel wįr umm krippą!«
Sturdörär fuor upp smą̊’tt, smą̊’tt. Tultą fikk sją̊ Ieðwig og Elį. Dier stuoð kwer og wild it far inn noð. »Kåm, Tultą«, lit dier. »Nų al du far iem!«
Rigåvur såg it estą og sliðą̊ min fuotjeð i, og dier lit fårå Tultų daitað krippum. »Kåm, Tultą«, sagd dier iessn og rak framm armą. Tultą ärd faðer enes sagde: »Kåm aut, lisslkullą mąi! Akk og ig irum jär, og du al fy uos
iemat.« Tultą, Ieðwig og Elį fuor daitað sliðum, so twerwart synliger. Mas djikk Tultun et my̨ ̨ötes,
lypteð upp en og sett nið ån i sliðån nest Akka. »Set ą̊ on oll warm klaut og djäv ener åv gą̊sum so Nimån stjikkeð minn«, lit an að Akk so tuog upp Tultų upi kninę og klemmd åna.
Rigåvur kam aut frą̊ bjärrę og stą’neð i dörum og bögles. Mas sett sig i sliðån, so an uld kenn sig dälan. An ugst rigåvur dugd dą̊fel it lypt upp boð estem og sliðåm min fuotjeð i. »Wiso tuogir Tultų minn ið?« spuord an og bögles ą̊ rigåvur so fuor att og framm fråmånað åm.

Ellåpt kapittleð
Mes sliðir add ferið gainum raiseð og war niðą̊ weem, so war grą’ll attrumsni’nn, sagd Mas að Ieðwig og Elin: »Tjär tokk fer jåpę. Um eddið it að fygt, wet ig it ur eddum ulað fårå fer tä fą̊ aut Tultų frą̊ bjärrę.«
»Men ur fuor dier fer tä fą̊ minn sig ån dait?« undres Ieðwig ą̊.
»Um dier irå mikkler so jåpas að auti raisę, werd dier starker, wet ig«, lit Mas. »Ur kam du upter bjärrę, Tultą?« spuord an etter.
»Ig flog«, suoräð ą̊. »Dier jält i nevum að mig, og sę kåy’tt dier umstað.« »Rigåvur wild naug läk min ið og, og gem ið under sniųo’nn«, sagd Mas wið Ieðwig og Elį. »Nų blås eð it noð mįer. Tolpt kapittleð
Eð wart mörka kweldn feld Wall og Tuov add stjuo’ssað iem kullum. Tuov wild fy og sją̊ war kullur byggde. Mas og Nimån og oll sjäks krippär fygd ą̊ i launggsliðåm attåni.
Wegen war it plyögd noð. Wall og Mas brukeð bytas umm plyö dar eð add sni’eð, men nų add estär faið wail ses dier add kumið iem frą̊ Rigåvbjärr.
Sparką kullum add Wall taið ą̊ släp eter sliðåm sainum. Boð Wall og Mas lyst sig min työrwiðim so dier add festað upą̊ sliðą̊. Eð stjien ien bit ini raiseð. Stjällur estum stjälleð.
»Luvið kumið jųot að sumbrem att!« yöpt Akke. »Eð ir mitjið wilum wais ið!« Kullur wend sig og nikkeð. »Ja, irið welkumner daitað uoss snjäst avið slutað skaulam«, sagd Tuova. Mes dier kam ini bynn, djärd Tuov iel skårån ųosynligan. Sliðir stą’neð fuost autą̊ gardem
nest Elin og sę nest Ieðwig. Indjin såg dier kamä. Kullur stieg yr og klemmd umm ollųiuop. Dier winkeð að weroðrum mes sliðir fuor atter ini raiseð.
Kullur stampeð autą̊ bru’nn sos dier brukeð, og mun og faðer og syskunę kam aut og yöpt til mes dier såg ukin so add kumið. Mes kullur add sagt åv wenn dier add weð minn um, lit mun og faðer að boð Ieðwig og Elin dier war faingner kullur add råkað so frekt noð fuok, so åp dyöm råð minn rigåvum.
»Welest dugdið fą̊ aut lislkullų frą̊ Rigåvbjärr! Men wenn ir eð fer fuok so byddj fer sig siuovum ini raisę og kunn glåmå min rigåvum og dugå djärå sig ųosynligų?«
